代码协作事项：【github】
1. 安装git
终端管理员运行：
winget install --id Git.Git -e --source winget

2.安装一个git GUI，例如：github desktop

3.注册/登录github账号

//Somnilight项目github网站：https://github.com/lain-wired0/Somnilight

4.本地克隆（clone）Somnilight项目库（repository）：https://github.com/lain-wired0/Somnilight.git

//目前库里共两个分支：
    main
    dev-a0
//main分支留作后续正式版整合，目前开发全部在dev-a0进行

5.后续每次在本地完成更改后，调试无误后可通过github desktop或vscode源码管理上传

6.github有时上传很慢，可以通过vpn代理解决，但是，直接开启vpn全局代理并不能让github也走代理，为了让github走代理，需要：

在终端管理员运行以下命令以配置github代理：
git config --global https.proxy "socks5://127.0.0.1:7890"
其中，7890是vpn代理服务的端口，每家服务的端口不一定一样，7890是clash的默认端口
PS：若终端报错“git不是可用的命令”，重启终端即可
